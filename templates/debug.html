Journal Management API - Debug Information

This page is only available in DEBUG mode.

Available API Endpoints

GET /api/ - API Root (List all endpoints)
GET /api/health/ - Health Check
GET POST /api/issues/ - Issues Management
GET /api/issues/recent-by-years/ - Recent Issues Grouped by Year
GET /api/issues/years-summary/ - Issues Years Summary
GET /api/issues/by-year/{year}/ - Issues by Specific Year
GET POST /api/articles/ - Articles Management
GET POST /api/authors/ - Authors Management
GET POST /api/keywords/ - Keywords Management
GET POST /api/news/ - News Management
GET POST /api/journals/ - Journals Management
GET POST /api/contact/ - Contact Messages
GET POST /api/board-members/ - Editorial Board Members
